ヘルメット調達

January 06, 2019

「使用開始から3年が目安」とされるメーカー推奨交換サイクルを忘れたわけではないが、 2008年7月 に調達したSHOEIのヘルメットも10年以上経過。

最近のモデルはサンバイザーがビルトインになっているという事もあって新調する方向で、初売りセールを期待して最寄りのバイク部品ショップ 福岡２りんかん に初出動。

自分用のは特に無かったが、レディースコーナーで HJC の CL-Y ソリッド や SPEEDPIT の「RJ-66」を物色。いずれもSG規格の全排気量対応で「RJ-66」ならは1万円以下と値頃ではあったものの、馴染みの無いブランドだったので持ち帰り検討しようとしていたところ、 ワイズギア の YJ-17 ZENITH (定価22,680円)のXSサイズが12,800円 になっていたのを発見。

現品限定で色もブラックだけだったが意外にも「黒でもいいよ♪」と好印象、ビルトインサンバイザーも「変身みたいでカッコいい」と気に入った様子。試着してみたところサイズ感も良さげだったので、ノーブランドより安心だろうという観点から調達決定。

ちなみにXSサイズの実測重量は1,458g。Lサイズで1,310gだった「YJ-5 ZENITH」よりも1割以上重くなってる勘定ですな。
